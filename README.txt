ГЛАВНЫЙ ЭКРАН

При запуске приложения пользователь видит главный экран со всеми доступными командами.


1. ДОБАВЛЕНИЕ КОНТАКТА

Для этого на главном экране выберите команду 1

Вводите имя (без специальных символов вроде "/", "-" и цифр).

Далее вводите фамилию или оставляете поле пустым.

Далее вводите номер телефона.

Если такого номера еще нет в БД, то контакт будет успешно добавлен.


2. ОТОБРАЖЕНИЕ СПИСКА ВСЕХ КОНТАКТОВ, ОТСОРТИРОВАННЫХ ПО ИМЕНИ

Для этого на главном экране выберите команду 2


3. ПОИСК КОНТАКТА ПО ЦИФРАМ В НОМЕРЕ ТЕЛЕФОНА

Для этого на главном экране выберите команду 3

Далее введите цифры, которые есть в искомом номере телефона

Вам будут показаны номера, имеющие в себе такую последовательность цифр


	3.1 ИЗМЕНЕНИЕ КОНТАКТА ПО НОМЕРУ ТЕЛЕФОНА
	
	Внутри функции поиска также есть функция изменения имени/фамилии контакта 	по номеру телефона

	Для этого вводим "да", когда появится строка "Желаете изменить данные 	контакта?"

	Затем вводите номер телефона необходимого контакта 

	Потом выбираете, что конкретно хотите изменить - имя или фамилию
	
	Далее вводите новое имя или фамилию


	3.2 УДАЛЕНИЕ КОНТАКТА ПО НОМЕРУ ТЕЛЕФОНА

	Также внутри функции поиска есть возможность удаления контакта по номеру 	телефона

	Для этого вводим "да", когда появится строка "Желаете удалить контакт?"

	Далее вводим номер телефона и контакт будет удален из таблицы базы данных


4. ВЫХОД ИЗ ПРОГРАММЫ

Обеспечивает завершение работы приложения
